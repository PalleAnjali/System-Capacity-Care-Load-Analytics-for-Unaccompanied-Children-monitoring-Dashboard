# System Capacity & Care Load Analytics for Unaccompanied Children

🔗 **Live Streamlit Dashboard:**
[![Open Dashboard](https://img.shields.io/badge/Streamlit-App-brightgreen)](https://careloadproject2.streamlit.app/)

## Project Description

This project analyzes system capacity and care load for unaccompanied children using CBP and HHS data. The dashboard visualizes intake trends, backlog indicators, and system load.

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly

## Features

* System Load Overview
* CBP vs HHS Load Comparison
* Net Intake & Backlog Trends
* KPI Summary Cards
